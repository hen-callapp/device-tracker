Connect to GitHub

This app stores its data in the private CallHub repo. Paste your personal access token (classic) to start.

  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Generate new token → Generate new token (classic)
  3. Check the repo scope (Full control of private repositories) → Generate

View-only access? A fine-grained token with read-only Contents permission on CallHub works too.
